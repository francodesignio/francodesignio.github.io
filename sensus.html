SENSUS

Visual Aid / Healthcare / Prototype

SENSUS is a spatial perception assistive glove for the visually impaired. It allows users to detect how close their hand is to objects before picking them up. It prevents accidents and facilitates grabbing objects.

When you turn on the SENSUS glove, the ultrasonic sensor will start to emit the signal. In case of detecting an object at a distance less than 50cm, the buzzer will start to give a small alarm indicating proximity. The closer the object is to the palm of the glove, where the sensor is located, the faster the buzzer will sound. Once the user has the object in hand, the emitted signal will detect the object and stop the buzzer.
